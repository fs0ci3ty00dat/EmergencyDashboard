# 🏫 תרגיל למידה מרחוק – בית ספר "אופק" אשדוד

דף אינטרנט אינטראקטיבי שנבנה עבור תרגיל למידה מרחוק בחירום, במסגרת **חודש הבינה המלאכותית: טבע ומדיה**.

## על הפרויקט

הדף מרכז את כל המידע הדרוש לצוות בית הספר ולמורים לניהול יום למידה מרחוק מסודר ויעיל. הוא כולל לוח משימות אינטראקטיבי לצוות הניהול, מדריך התחברות טכני לתלמידים, ו-6 מערכי שיעור מותאמים לגילאים – כיתות א' עד ו' – סביב נושא הבינה המלאכותית והטבע.

## תכולה

- **לוח ניהול** – משימות מסומנות לפי תפקיד (מנהלת / רכזת תקשוב)
- **מדריך התחברות** – הנחיות שלב-אחר-שלב להתחברות דרך פורטל משרד החינוך
- **מערכי שיעור** – כל שכבה מקבלת שיעור ייחודי הכולל: משימת טרום שיעור, מהלך השיעור, תוצר להגשה, כרטיס יציאה ומשימה לאחר השיעור

## טכנולוגיות

בנוי עם HTML, CSS ו-JavaScript טהורים – ללא תלות בספריות חיצוניות. מתאים לצפייה בדפדפן בלבד.
